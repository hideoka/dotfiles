# dotfiles

## install requirement
- make
- git
- HackGen Console NF font
- alacritty
- neovim
- fzf
- rust
- Homebrew(macOS)

## make command
```
$ git clone https://github.com/hideoka/dotfiles.git
$ cd dotfiles
$ make init
```

## create symbolic link
create symbolic link to dotfiles
```
$ make deploy
```
